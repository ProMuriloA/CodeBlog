HTML, the foundational building block

As you start your coding journey you will face a lot of subjects, some are mandatory, some are not, it depends on your goals. But, the first step is of utmost importance. And the first step for a software developer is HTML.

Html stands for HyperText Markup Language and it is used to print text, images and content in general to a webpage in a website, it uses tags and and it has its own syntax, it is not a programming language but it is needed to create a webpage, without it, no content would show when you open a site. It is the "skeleton" of a webpage.

However, for the current webpages to run it is needed much more than just HTML. Which bring us to the next topic: CSS.

CSS, adding style to a webpage

Without CSS webpages would be boring and colourless. CSS stands for Cascading Style Sheets and it is used for adding style to webpages. By creating rulesets and pairs of attribute and values. Furthermore, with CSS you can change the way pages look, moving elements of HTML from one place to the other, adding animations and transitions and giving the webpage a fun look so it can be shown to the user. By the way: I'm sorry, but CSS is not a programming language.

You may be thinking: "That's it?". No, that's not it. As you've seen, there is a lot of content we've got to cover. So, let's go ahead into our next topic (and a real programming language): Javascript.

Javascript, that's where the fun actually begins

Javascript is a programming language. While learning it you will face other challenges, such as programming logic, which encompasses: loops, conditionals, variables, functions, data types, classes and other "interesting" stuff (it is only interesting if you like computers). Then you will learn how to do a lot of stuff with it, such as importing modules (you gotta learn what modules are first), doing nice mathematical stuff, creating stuff no one is gonna use (unless they really like computers and Mathematics), and then after all this you are going to learn actual coding stuff, which bring us to the next topic: Frameworks.

Frameworks, lots of them!

There are a lot of frameworks for a lot of different purposes, but they all have something in common: They make coding easy, maintanable and cut a lot of unnecessary repetition. You will find many frameworks on your way to get a Front-end job. Make good use of them, you're gonna use it all the time.

Data Structures, beware!

Data Structures is the most complex of all, it involves mathematics and uses a programming language, such as JavaScript or Python, it is an important subject to study beacause it can make your website be fast or slow, it can make the website have problems or not. The correct use of Data Structures makes websites work properly, the incorrect use of Data Structures can make the website work in a wrong way or not work at all.
